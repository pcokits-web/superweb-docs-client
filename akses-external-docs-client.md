## â“ 11) FAQ

**Q: Mengapa saya tidak bisa mengunggah dokumen?**  
A: Akun Client bersifat **View Only**. Kirim dokumen ke PIC PCO Kits untuk diunggah.

**Q: Saya tidak melihat folder tertentu.**  
A: Akses mungkin **Restricted**. Minta PIC menambahkan perusahaan Anda pada **Manage Access**.

**Q: Kapan Service Report bulan berjalan tersedia?**  
A: Setelah pekerjaan selesai & diverifikasi. Tanyakan jadwal ke PIC proyek Anda.

**Q: Bolehkah membagikan tautan file ke pihak luar?**  
A: Sebaiknya **tidak**. Akses hanya untuk akun terdaftar. Minta PIC menambahkan user yang berwenang.
