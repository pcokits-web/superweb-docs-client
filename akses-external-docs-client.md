## â“ 12) FAQ

**Q: Mengapa saya tidak bisa mengunggah dokumen?**  
A: Akun Client bersifat **View Only**. Kirim dokumen ke Admin PCO Kits untuk diunggah ke sistem.

**Q: Saya tidak melihat folder tertentu.**  
A: Kemungkinan akses Anda **Restricted**. Minta Admin menambahkan perusahaan Anda pada pengaturan **Manage Access**.

**Q: Apakah saya boleh membagikan tautan file ke pihak luar?**  
A: **Tidak diperbolehkan.** Akses hanya untuk akun yang terdaftar di sistem. Hubungi Admin jika membutuhkan bantuan berbagi file secara resmi.
